# Luxe Nails Studio - Simple Nail Salon Website

A clean, beginner-friendly website for a mock nail salon in Orlando, Florida. Built with pure HTML, CSS, and JavaScript—no frameworks or external dependencies needed!

## 📚 About This Study Project

**This is a learning and experimental project created with GitHub Copilot.** It was built to:
- Study and try out GitHub Copilot's code generation capabilities
- Learn web development fundamentals (HTML, CSS, JavaScript)
- Understand how AI assistants can help in building websites

**Important:** This is a **mock/demo application** for educational purposes only. The salon name, address, phone number, services, pricing, and all business information are **completely fictional and not real**. This is not an actual operating business.

The HTML structure, CSS styling, and JavaScript functionality were created with the assistance of GitHub Copilot, making it a great example of how modern AI tools can accelerate web development learning.

## Project Overview

**Luxe Nails Studio** is a demonstration website showcasing what a small nail salon might look like online. It includes:

- 🏠 **Hero Section** - Eye-catching header with a beautiful nail salon image
- ℹ️ **About Section** - Brief information about the salon
- 💅 **Services Section** - All common nail salon services with pricing
- 🕐 **Business Hours** - Operating hours for each day of the week
- 📞 **Contact Section** - Address, phone number, and a simple contact form
- 📱 **Responsive Design** - Looks great on phones, tablets, and desktop computers

## Files in This Project

- **index.html** - The main HTML structure of the website
- **styles.css** - All styling and responsive layout (no inline styles!)
- **script.js** - JavaScript for interactivity (smooth scrolling, form handling)
- **README.md** - This file

## How to Run Locally

### Easy Way: Just Open the HTML File

1. Download or clone this repository to your computer
2. Right-click on `index.html`
3. Select "Open with" → Your web browser
4. Enjoy! 🎉

Or simply double-click `index.html` if your computer is set up to open HTML files in your default browser.

### Alternative: Using a Local Server

If you want to use a local web server (optional):

**Using Python 3:**
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.

**Using Node.js (with http-server):**
```bash
npx http-server
```

## Features Explained

### Navigation Bar
- Sticky navigation that stays at the top when scrolling
- "Book Now" button that smoothly scrolls to the Contact section

### Services
- 8 common nail salon services displayed in a clean grid
- Each service shows a description and price
- Cards have a hover effect that lifts them up slightly

### Contact Form
- Simple form with Name, Email, and Message fields
- Basic validation to ensure all fields are filled
- Email format validation
- Shows a success message when submitted
- Form automatically resets after submission

### Responsive Design
- Works perfectly on mobile phones (320px and up)
- Tablet-friendly layouts (768px and up)
- Beautiful desktop experience (1000px+)
- Touch-friendly buttons and spacing
